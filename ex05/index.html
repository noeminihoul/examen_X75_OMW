Encore là ? Bon, j'ai un pote qui a créé un générateur d'étiquette de pot de miel. J'essaye de détourner son code pour en faire un générateur de memes. J'arrive à télécharger un meme mais il me manque quelques fonctionnalités. Tu sais m'aider ?

  • J'ai fait appel à une API pour récupérer une image au hasard mais je n'arrive pas à l'afficher :(
  • J'aimerais rajouter une deuxieme ligne de texte
  • J'aimerais changer la taille du texte mais j'ai essayé un truc qui fonctionne pas
  • Ca serait super de pouvoir aussi changer la couleur du texte
  • Le bouton pour sauvegarder en JPEG fonctionne, mais renverra une erreur tant qu'on n'arrivera pas à afficher l'image

D'après ChatGPT, il faut faire ce qui est marqué en-dessous.

JE TE CONSEILLE D'OUVRIR LE DOSSIER 'ProjectReact' DANS UNE NOUVELLE FENETTRE DE VSCODE. 

  1. Assurez-vous d'avoir installé Node.js et npm (gestionnaire de paquets Node.js) sur votre ordinateur. Si ce n'est pas
  le cas, vous pouvez les télécharger et les installer à partir de https://nodejs.org/.

  2. Ouvrez une invite de commandes (ou un terminal, si vous êtes sur Mac ou Linux) et accédez au répertoire du projet
  React existant.

  3. Installez les dépendances du projet en exécutant la commande npm install ou yarn install, selon que vous utilisez
  npm ou Yarn comme gestionnaire de paquets.

  4. Une fois que les dépendances sont installées, vous pouvez démarrer le serveur de développement en exécutant la
  commande npm start ou yarn start. Cela lancera un serveur local qui exécutera votre application React et vous
  permettra de la visualiser dans votre navigateur à l'adresse http://localhost:3000/.
